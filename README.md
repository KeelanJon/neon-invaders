# Neon Invaders DX

A mobile-friendly retro Space Invaders-style arcade game built with React, TypeScript, Vite, and a canvas renderer.

## Features

- Pixel-art canvas rendering
- Touch controls for mobile
- Keyboard controls for desktop
- Powerups: shield, rapid fire, spread, laser, drones, chain lightning, chrono, magnet, overdrive, singularity
- Boss waves
- Rift-zone waves
- Meteors, destructible shields, combo scoring, perfect-wave bonuses
- Local high score
- GitHub Actions build check

## Run locally

```bash
npm install
npm run dev
```

Open the local URL Vite prints in your terminal.

## Test from Codespaces

Run the dev server in the Codespace, then open the forwarded port `5173` URL on desktop or mobile.

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

After pushing this repo to GitHub, you can deploy with Vercel, Netlify, or GitHub Pages. For GitHub Pages with Vite, set the repo Pages source to a build workflow or use a Pages action.
